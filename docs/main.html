Main module of the game.
 
This script launches the graphical application, manages the main game loop,
the initialization of data (cities, tracks, destinations, cards), player management
(humans and AIs), as well as rendering and user interactions.
 
Main Features :
- Loading board and card data from CSV files.
- Game session management: turns, drawing, purchasing tracks, objectives, scores.
- Graphical interface with Pygame (displaying the board, cards, buttons, etc.).
- Support for multiple types of AI and human players.
- Display of the selection screen, the active game, and the end-game screen.
 
Main Class :
    Jeu : Management of game logic and rendering.
 
Execution :
    Run this script to start the game with the graphical interface.
