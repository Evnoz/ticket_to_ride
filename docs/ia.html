Module ia
=========
 
This module defines different AI classes for the game.
Each AI inherits from the Joueur class and implements a different strategy for automated play:
- IA_aleatoire : chooses its actions randomly.
- IA_capitaliste : prioritizes buying the most expensive track possible.
- IA_objectif : prioritizes completing its destination objectives using shortest-path algorithms.
 
Main Functions :
- Management of automated actions for an AI player (drawing, buying, planning).
- Graph algorithms for objective planning (Dijkstra).
- Utility methods for buying tracks and managing cards.
 
Classes :
    IA : Base class for an AI.
    IA_aleatoire : AI that plays randomly.
    IA_capitaliste : AI that maximizes the value of its purchases.
    IA_objectif : AI that optimizes the completion of its objectives.
 
Authors : Alice DEBELS and Evann MICHEL
