Module joueurs
==============
 
This module defines the Joueur class, which manages all features related to a player
in the game.
 
Main Features :
- Management of cards, tracks, objectives/destinations, and the player's score.
- Methods to add/remove cards, update the score and remaining train cars.
- Graph algorithms (BFS, DFS) to validate objectives and calculate the longest built path.
 
Class :
    Joueur : Represents a game player with their associated properties and methods.
 
Authors : Alice DEBELS et Evann MICHEL
